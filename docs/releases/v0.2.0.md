# v0.2.0 Release Notes

**Release Date:** 2026-06-29

## Highlights

- Interactive D3.js web visualization for standards catalogs
- New CLI commands: `export` and `generate-web`
- Full MkDocs documentation site

## New Features

### Web Visualization

Interactive browser-based visualization for exploring standards catalogs:

- **Graph View**: Force-directed network showing standards and their relationships
- **Table View**: Sortable table with all standard details
- **Filtering**: Filter by layer, organization, category, and status
- **Color-coded**: Standards colored by identity layer
  - Human: Cyan
  - Agent: Orange
  - Workload: Purple
  - Service: Green
- **Interactive**: Click nodes to open specifications, drag to rearrange, zoom/pan

### New CLI Commands

#### `export`

Export catalog as formatted JSON:

```bash
standards-catalog export catalog.yaml > catalog.json
```

#### `generate-web`

Generate a standalone HTML visualization with embedded catalog data:

```bash
standards-catalog generate-web catalog.yaml -o standards-graph.html
```

The generated HTML file can be:

- Opened directly in a browser
- Embedded in MkDocs via iframe
- Hosted on any static file server

### MkDocs Documentation Site

Full documentation site with Material theme:

- Home page with overview and quick start
- Getting Started guide
- CLI Reference
- Library Usage examples
- Schema Reference
- Interactive Visualization page
- Changelog

## Installation

```bash
go install github.com/grokify/standards-catalog-framework/cmd/standards-catalog@v0.2.0
```

## Usage Examples

### Generate Visualization

```bash
# Generate HTML visualization
standards-catalog generate-web catalog.yaml -o docs/standards-graph.html

# Export as JSON
standards-catalog export catalog.yaml > catalog.json
```

### Embed in MkDocs

```markdown
<iframe src="standards-graph.html" width="100%" height="700" style="border:none;"></iframe>
```

Or link directly:

```markdown
[View Standards Graph](standards-graph.html)
```
